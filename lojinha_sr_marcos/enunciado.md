# Lojinha do seu marcos

*Crie um sistema que possa: ( Criar | Ler | Editar | Deletar ) produtos de uma lojinha.*

O programa mesmo que reiniciado deve ser capaz de manter os dados da sessão anterior. O programa também deve oferecer uma interface por terminal para o usuário escolher sua ação, podendo ser:

- 1 → Criar
- 2 → Ler
- 3 → Editar
- 4 → Deletar
- 5 → Sair

O programa ao oferecer as escolhas para o usuário, deve antes informar se existe algum produto com menos de 3 exemplares.

 **(Criar)**
 Os produtos devem ser registrados por:
→ identificador único e aleatório;
→ nome do produto;
→ quantidade do produto.

**Ps**: Não deverá ser possível adicionar um item que já exista.

 **(Ler)**
Usuário deve escolher o filtro que ele quer utilizar para ler os produto, podendo ser:
→ Todos os itens;
→ Nome;
→ Id;
→ Quantidade.

**Ps**: Para ler o usuário deve fornecer o nome do produto ou o id do produto ou a quantidade do produto.

**(Editar)**
Usuário deve escolher o filtro que ele quer utilizar para escolher o produto que ele quer editar, podendo ser:
→ Nome;
→ Id.

**Ps**: Após escolher o item as informações sobre o mesmo deverão aparecer na tela e o usuário deverá escolher entre as opções:
→ Editar nome;
→ Editar quantidade.

**(Deletar)**
Usuário deve escolher o filtro que ele quer utilizar para escolher o produto que ele quer deletar, podendo ser:
→ Nome;
→ Id.
